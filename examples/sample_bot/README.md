# mper サンプルBot

mperの動作確認用のシンプルなdiscord.py Botです。

## ファイル構成

| ファイル | 説明 |
|----------|------|
| `bot.py` | サンプルBot本体（モデレーション・ユーティリティコマンド） |
| `generate_invite.py` | mperを使って招待URLを生成するスクリプト |

## 使い方

### 1. 招待URLを生成する

```bash
# mperをインストール
pip install mper

# 招待URLを生成
python generate_invite.py YOUR_CLIENT_ID
```

## Botの機能

### モデレーションコマンド
- `/kick` - メンバーをキック
- `/ban` - メンバーをBAN
- `/purge` - メッセージを一括削除

### ユーティリティコマンド
- `/ping` - Botの応答速度を確認
- `/say` - メッセージを送信

## Botを実行する場合

```bash
export DISCORD_TOKEN="your_bot_token"
python bot.py
```

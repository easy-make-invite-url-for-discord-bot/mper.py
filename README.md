# mper.py
for discord.py bot

## インストール方法
```sh
pip install mper
or
pip3 install mper
```

## 使い方
ターミナルまたはシェルで
```sh
mper /path/to/projects/ client_id
```
例
```sh
mper /user/freewifi/test_bot/ 12345678990
```
